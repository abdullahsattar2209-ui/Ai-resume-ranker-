Task Documentation · AI-2

  
TalentMatch AI — Resume Screening & Candidate Ranking System

  
An AI-powered system for Task AI-2 of the TEYZIX CORE Internship Program (June Batch 2026). Automatically analyzes resumes, extracts candidate information, evaluates suitability against job requirements, and produces a ranked, explainable shortlist.


  
Why The PDF Upload Now Works

  
Previous versions failed because the file input either had no real PDF parser, or was hidden behind a JavaScript .click() trigger that gets blocked in sandboxed preview environments. This version fixes both problems:

  
    
    
  • The file input is fully visible — styled with native ::file-selector-button CSS, no hidden elements, no JS-triggered clicks.
    • 
    
  • PDF.js (Mozilla's real PDF rendering engine) is loaded directly from CDN and used to extract text from each uploaded PDF page by page — this is the same library browsers use internally to display PDFs.
    • 
    
  • Drag & drop is wired as a genuine backup, not the only path.
    • 
    
  • Each file shows live status (loading → success/error) so you always know what happened.
    •
